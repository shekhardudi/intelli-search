# intelli-search
AI-powered intelligent company search engine built for 7M+ records. Automatically classifies queries and routes them through BM25 lexical, kNN semantic (384-dim vectors + Reciprocal Rank Fusion), or agentic search (GPT-4o + LangChain + live web lookup). Built with FastAPI, React, OpenSearch, Redis, and full OpenTelemetry observability.
